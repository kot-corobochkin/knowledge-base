## 6. Pattern Recognition

Pattern recognition is the process of classifying objects based on their attributes and finding matches between new stimuli and stored patterns.
- **Process:** It involves assigning a label to a given input value. For example, classifying an email as "spam" or "non-spam."
- **Learning Approaches:**

**Supervised Learning:** The algorithm is trained on a set of data that has been manually labeled with the correct output.
- **Unsupervised Learning:** The algorithm is trained on data that has not been labeled. It attempts to find inherent patterns in the data on its own.
- **Semi-Supervised Learning:** A combination that uses a small set of labeled data with a large amount of unlabeled data.
